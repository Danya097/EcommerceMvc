# Ecommerce MVC Project

This project is an ASP.NET Core MVC application created to practice source control, GitHub pull requests, and Bootstrap 5.

## Description
This project demonstrates:
- ASP.NET Core MVC structure
- Git and GitHub workflow (branches, pull requests)
- Bootstrap 5 layout and components
- GitHub Actions for build validation

## Getting Started
1. Clone the repository
2. Open the solution in Visual Studio 2022
3. Run the project using IIS Express or the project profile

## Technologies Used
- ASP.NET Core MVC
- C#
- Bootstrap 5
- GitHub Actions
