# Coding Resources for .NET Developers

## Overview
This markdown file contains a collection of useful resources for learning and working with .NET.
It was created as part of a class assignment to demonstrate the use of Markdown in a GitHub project.

## .NET Learning Resources
- Microsoft Learn — official .NET documentation and tutorials
- C# Documentation — language reference and examples
- Stack Overflow — community questions and answers
- W3Schools — beginner-friendly tutorials

## Development Tools
- Visual Studio
- Visual Studio Code
- SQL Server Management Studio
- GitHub

## Coding Best Practices
- Use meaningful variable and method names
- Follow C# naming conventions
- Comment complex logic
- Keep methods short and focused
- Use version control (Git)

## Why Markdown?
Markdown is a lightweight markup language that allows developers to create clean and readable
documentation directly inside GitHub repositories.
